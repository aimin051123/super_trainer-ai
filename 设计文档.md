## 4. 数据库设计

### 4.1 ER 关系

```
course (1) ────< (N) knowledge_point
course (1) ────< (N) quiz_detail
course (1) ────< (N) chat_history
```

### 4.2 初始化 SQL

```sql
CREATE TABLE IF NOT EXISTS course (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    description TEXT DEFAULT '',
    exam_date TEXT DEFAULT '',
    daily_study_minutes INTEGER DEFAULT 240,
    created_at TEXT DEFAULT (datetime('now','localtime'))
);

CREATE TABLE IF NOT EXISTS knowledge_point (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    course_id INTEGER NOT NULL,
    name TEXT NOT NULL,
    subject TEXT DEFAULT '',
    mastery REAL DEFAULT 0.0,
    total_count INTEGER DEFAULT 0,
    correct_count INTEGER DEFAULT 0,
    is_weak INTEGER DEFAULT 0,
    weak_reason TEXT DEFAULT '',
    last_reviewed TEXT DEFAULT '',
    FOREIGN KEY (course_id) REFERENCES course(id) ON DELETE CASCADE
);

CREATE TABLE IF NOT EXISTS quiz_detail (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    course_id INTEGER NOT NULL,
    page_name TEXT NOT NULL,
    subject TEXT DEFAULT '',
    question TEXT NOT NULL,
    user_answer TEXT DEFAULT '',
    correct_answer TEXT NOT NULL,
    is_correct INTEGER DEFAULT 0,
    difficulty TEXT DEFAULT '基础',
    timestamp TEXT DEFAULT (datetime('now','localtime')),
    FOREIGN KEY (course_id) REFERENCES course(id) ON DELETE CASCADE
);

CREATE TABLE IF NOT EXISTS chat_history (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    course_id INTEGER NOT NULL,
    role TEXT NOT NULL,
    content TEXT NOT NULL,
    timestamp TEXT DEFAULT (datetime('now','localtime')),
    FOREIGN KEY (course_id) REFERENCES course(id) ON DELETE CASCADE
);
```

### 4.3 关键查询

**薄弱点扫描（规则1：正确率 < 60% 且 ≥3 次）**：
```sql
SELECT * FROM knowledge_point
WHERE course_id = ? AND total_count >= 3 AND mastery < 0.6;
```

**最近 N 次答题记录（规则2/3 使用）**：
```sql
SELECT * FROM quiz_detail
WHERE course_id = ? AND page_name = ?
ORDER BY timestamp DESC LIMIT 5;
```

**某课程掌握度汇总**：
```sql
SELECT subject, COUNT(*) AS point_count, AVG(mastery) AS avg_mastery
FROM knowledge_point WHERE course_id = ?
GROUP BY subject;
```

**最近 5 轮对话历史**：
```sql
SELECT role, content FROM chat_history
WHERE course_id = ?
ORDER BY timestamp DESC LIMIT 10;
-- 取最近 10 条消息（5 轮 = 5 user + 5 assistant）
```

### 5.2 统一调用函数

```python
import openai

def call_ai(prompt, system_prompt=None, json_mode=False, stream=False):
    """
    统一 AI 调用入口
    - prompt: 用户消息（纯文本或指令前缀）
    - system_prompt: System Message（角色 + 文档内容 + 掌握度数据）
    - json_mode: True → response_format={"type":"json_object"}
    - stream: True → 返回生成器，逐 token yield
    """
    client = openai.OpenAI(api_key=api_key, base_url=base_url)
    messages = []
    if system_prompt:
        messages.append({"role": "system", "content": system_prompt})
    messages.append({"role": "user", "content": prompt})

    kwargs = {
        "model": model,
        "messages": messages,
        "temperature": 0.7,  # 问答用 0.7；出题用 0.9
        "max_tokens": 4096,
        "timeout": 30,
    }
    if json_mode:
        kwargs["response_format"] = {"type": "json_object"}
    if stream:
        kwargs["stream"] = True

    response = client.chat.completions.create(**kwargs)
    if stream:
        return (chunk.choices[0].delta.content or "" for chunk in response)
    return response.choices[0].message.content
```

### 5.3 各功能的调用参数

| 功能 | system_prompt | json_mode | stream | temperature |
|---|---|---|---|---|
| 学科分类 | "判断文本所属学科" | ✅ | — | 0.3 |
| 碎片类型识别 | "判断笔记类型和标题" | ✅ | — | 0.3 |
| 知识编译 | "以下学习资料，整理为结构化页面" | ✅ | — | 0.7 |
| 日常问答 | 角色设定 + 课程文档 + 掌握度数据 | — | ✅ | 0.7 |
| 出题 | 角色设定 + 知识库内容 + 掌握度数据 | ✅ | — | 0.9 |
| 批改 | "批改以下答案" | ✅ | — | 0.1 |
| 复习计划 | 角色设定 + 考试日期 + 掌握度 + 遗忘曲线参数 | — | — | 0.7 |

### 5.4 Streamlit SSE 逐字显示实现

```python
# 问答时，用 st.empty() 占位，逐 token 覆盖渲染
def stream_answer(prompt, system_prompt):
    placeholder = st.empty()
    full_text = ""
    for token in call_ai(prompt, system_prompt, stream=True):
        full_text += token
        placeholder.markdown(full_text + "▌")  # 光标效果
    placeholder.markdown(full_text)  # 最终文本
    return full_text
```

## 7. 文件处理链路

### 7.1 文件解析

```python
def parse_file(uploaded_file):
    name = uploaded_file.name.lower()
    if name.endswith(".pdf"):
        from pypdf import PdfReader
        reader = PdfReader(uploaded_file)
        return "\n".join(p.extract_text() or "" for p in reader.pages)
    elif name.endswith((".ppt", ".pptx")):
        from pptx import Presentation
        prs = Presentation(uploaded_file)
        texts = []
        for slide in prs.slides:
            for shape in slide.shapes:
                if shape.has_text_frame:
                    texts.append(shape.text_frame.text)
        return "\n".join(texts)
    elif name.endswith((".doc", ".docx")):
        from docx import Document
        doc = Document(uploaded_file)
        return "\n".join(p.text for p in doc.paragraphs)
    elif name.endswith((".txt", ".md")):
        return uploaded_file.read().decode("utf-8")
```

### 7.2 文本切片

标题识别正则（优先级从高到低）：

| 正则 | 匹配示例 |
|---|---|
| `^#{1,4}\s+.+` | `## 二叉树遍历` |
| `^\d+[\.\、\)]\s*.+` | `1. 概述`、`3、链表` |
| `^第[一二三四五六七八九十\d]+[章节].+` | `第一章 绪论`、`第2节 排序` |

切片策略：
- 匹配到标题 → 新建一个片段，用标题作为片段名
- 全文只切出 ≤1 段 → 降级按空行分段（`\n\n` 分割）

## 8. 薄弱点判定算法

```python
def judge_weak(page_name, mastery, total_count, last_is_correct):
    """
    输入：知识点名、当前正确率、总答题次数、本次是否答对
    输出：(is_weak: bool, reason: str)
    """
    recent = get_recent_answers(page_name, n=5)  # 最近 5 次记录

    # 规则4：数据不足 + 首次错误 → 待观察
    if total_count < 3 and not last_is_correct:
        return False, "待观察"

    # 规则2：连续错误 ≥2 次
    consecutive = 0
    for r in recent:
        if r.is_correct:
            break
        consecutive += 1
    if consecutive >= 2:
        return True, f"连续错误{consecutive}次"

    # 规则1：≥3 次答题 + 正确率 < 60%
    if total_count >= 3 and mastery < 0.6:
        return True, f"答题{total_count}次正确率{mastery:.0%}，低于60%"

    # 规则3：总体可以但近期下滑
    if total_count >= 5 and mastery >= 0.6 and len(recent) >= 3:
        recent_3 = recent[:3]
        recent_mastery = sum(1 for r in recent_3 if r.is_correct) / 3
        if recent_mastery < 0.4:
            return True, f"总正确率{mastery:.0%}但近3次仅{recent_mastery:.0%}"

    return False, ""
```

## 11. 风险与应对

| 风险 | 影响 | 应对 |
|---|---|---|
| DeepSeek API 限流 | 高频调用被拒绝 | 客户端加请求间隔 ≥1s；出题失败时提示"请稍后重试" |
| 文档过长超出 context window | 截断导致回答不完整 | 统一截取 4000 字符；编译后知识库每页控制在 2000 字内 |
| PDF 解析兼容性 | 部分 PDF 提取为空或乱码 | 多层回退：pypdf 失败 → 尝试 pdfplumber → 提示用户该文件无法解析 |
| AI 返回 JSON 解析失败 | 出题/编译流程中断 | try/except + 错误提示 + 重试按钮；学科分类回退到关键词匹配 |
| Streamlit 首次运行 email 提示 | 阻塞启动 | `.streamlit/config.toml` 设置 `browser.gatherUsageStats = false` |
| 网络代理问题 | pip 安装失败 | 文档中注明代理配置方法；提供离线依赖包备选方案 |
