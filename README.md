## 项目简介

SuperTutor 是一个面向大学生的期末复习辅助系统。它解决了四个痛点：

| 痛点 | SuperTutor 的方案 |
|---|---|
| 📄 资料散乱，缺乏统一管理 | 上传 PDF/PPT/Word → AI 自动分类、切片、提取知识点 |
| ❓ 不知道自己的薄弱环节 | 刷题后自动分析，4 条规则精准定位薄弱点 |
| 📅 复习缺乏计划性 | 输入考试日期 → AI 结合艾宾浩斯遗忘曲线生成每日计划 |
| 🤖 直接问 ChatGPT 答案不可靠 | 基于你自己的课件回答，标注来源页面 |

**一句话**：你给它课件，它拆成知识点；你刷它出的题，它告诉你哪里弱；你告诉它哪天考试，它排一份每天干什么的计划。

## 核心功能

### 📚 知识收割
- 上传 PDF / PPT / Word / TXT / Markdown 文件
- 自动提取文字 → 按章节标题切片 → AI 判断学科 → 存入知识库
- 上传后 **AI 自动提取结构化知识点**，实时显示在侧边栏

### ✍️ 碎片知识收割
- 手动输入易错点、顿悟笔记
- AI 自动识别学科和实体类型（概念 / 算法 / 对比 / 考试技巧）

### 🔄 知识编译
- 将原始碎片 AI 重写为格式统一、层次分明的知识页面
- 自动建立 `[[双向链接]]`，形成知识网络
- 自动更新全局索引和变更日志

### ❓ 知识问答
- 基于你的资料回答（先检索知识库，再调 AI）
- 支持 **多轮对话**（保留最近 5 轮上下文）
- **SSE 流式输出**（逐字显示，不用等）
- 答案标注引用来源页面

### ✏️ 精准刷题
- AI 基于你的知识库生成选择题（4 选项，含答案和解析）
- 支持选择学科、难度（基础/中等/困难/自动匹配）、题量（3/5/8/10）
- **薄弱点优先**：自动多出你不擅长的知识点
- 临时选中 → 点击"提交"确认 → 自动批改（本地比对，不调 AI）
- 每道题关联知识点 ID（外键绑定，非字符串匹配）

### 🔍 薄弱点判断（4 条规则）
- 规则 1：答题 ≥3 次，正确率 < 60% → 薄弱
- 规则 2：连续答错 ≥2 次 → 薄弱
- 规则 3：总正确率 ≥60%，但最近 3 次 < 40% → 薄弱
- 规则 4：答题 < 3 次且首次错误 → 待观察

### 📕 错题本
- 所有答错的题自动收录
- 支持 **重新作答**（隐藏正确答案）
- 答对 → 标记已掌握，从列表消失
- 答错 → 复习次数 +1，继续保留

### 📅 复习规划
- 设置考试日期和每日学习时长（分钟）
- AI 结合 **艾宾浩斯遗忘曲线**（第 1/2/4/7/15 天间隔复习）
- 薄弱点优先安排，最后 3 天总复习 + 错题回顾
- 掌握度进度条（🟢 >80% / 🟡 50-80% / 🔴 <50%）
- 测验历史趋势分析
- 支持下载计划为 Markdown 文件

## 快速开始

### 环境要求

- Python 3.8+
- DeepSeek API Key（[免费注册获取](https://platform.deepseek.com)）

### 安装

```bash
# 1. 克隆项目
git clone https://github.com/aimin051123/super-tutor.git
cd super-tutor

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置 API Key
# 复制配置模板并填入你的密钥
cp config.properties.example config.properties
# 编辑 config.properties，将 sk-your-key-here 替换为真实密钥
```

### 启动

```bash
cd 代码
streamlit run app.py
```

浏览器自动打开 `http://localhost:8501`。

> **首次使用**：侧边栏会自动创建"默认课程"。上传一份课件 → 点击"知识编译" → 开始刷题。

## 使用指南

### 基础流程

```
① 上传课件（拖入 PDF/PPT/Word）
    ↓
② 点击"知识编译"（AI 整理知识库）
    ↓
③ 去刷题页选学科和难度，生成题目
    ↓
④ 提交 → 自动批改 → 查看薄弱点
    ↓
⑤ 去复习规划页生成每日计划
```

### 日常循环

```
刷一组题 → 系统标薄弱点 → 下次出题自动多出薄弱题
    ↓
薄弱点补上 → 自动降级 → 错题本复习 → 看计划是否更新
```

### 多课程使用

侧边栏点击"➕ 新建课程" → 输入课程名 → 切换课程 → 上传该课程的课件。每门课程的数据（知识点、答题记录、错题、计划）完全独立。

## 技术栈

| 层级 | 技术 | 说明 |
|---|---|---|
| 网页框架 | **Streamlit** 1.28+ | 纯 Python 写网页，零前端代码 |
| AI 大模型 | **DeepSeek-chat** | 通过 OpenAI 兼容协议调用 |
| AI SDK | **openai** 1.0+ | 自带重试、超时、SSE 解析 |
| 数据库 | **SQLite** 3 | 单文件数据库，零配置，拷贝即迁移 |
| PDF 解析 | **pypdf** 3.0+ | 本地解析，无需网络 |
| PPT 解析 | **python-pptx** | 提取幻灯片文本 |
| Word 解析 | **python-docx** | 提取段落文本 |
| 配置管理 | **configparser** | 读取 .properties 文件 |

### 关键设计决策

| 决策 | 选了 | 没选 | 原因 |
|---|---|---|---|
| 网页框架 | Streamlit | Flask/Django | 无需写 HTML/CSS/JS |
| 数据库 | SQLite | MySQL | 无需安装服务，拷贝文件夹即迁移 |
| 出题类型 | 选择题 | 判断/填空/简答 | 批改最简单可靠（字符串比对） |
| 检索方式 | 关键词匹配 | 向量数据库 | 几百个页面无需向量 |
| 薄弱判定 | 4 条 if-else | AI 判断 | 规则简单，AI 不可控 |
| 批改 | 本地比对 | AI 批改 | 出题时 AI 已返回答案 |
| 代码风格 | 纯函数、单文件 | 面向对象、多文件 | 零基础上手 |

## 配置说明

### `config.properties`

```ini
[DEFAULT]
DEEPSEEK_API_KEY=sk-你的密钥
DEEPSEEK_BASE_URL=https://api.deepseek.com
DEEPSEEK_MODEL=deepseek-chat
DEEPSEEK_TIMEOUT=30
```

### API Key 读取优先级

1. **侧边栏手动输入**（实时生效，重启后丢失）
2. **环境变量** `DEEPSEEK_API_KEY`
3. **config.properties**（持久化配置）

三种方式同时支持，程序自动按优先级选择。
